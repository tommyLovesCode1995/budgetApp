# Mini Budget (No Build)

A tiny budget tracker you can run locally. Everything is saved in your browser via `localStorage` (no server).

## Run
Open `index.html` in your browser.

## Features
- Add income + expenses with date + category
- Totals: Income, Expenses, Balance
- Search + filter by type
- Delete single rows
- Export CSV
- Clear all (local only)
- Light/Dark theme toggle (saved)

## Files
- `index.html`
- `styles.css`
- `script.js`
